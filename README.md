# Tên dự án
Mục đích:To-do app with Pomodoro

## Tech Stack
- Node.js + ExpressJS
- MongoDB + Mongoose
- React + Tailwind+ Shadcn(frontend)
- JWT Auth

## Cấu trúc thư mục chính
src/
 ├── controllers/
 ├── services/
 ├── models/
 ├── middlewares/
 ├── routes/
 ├── utils/
 └── app.js

## Cách chạy
1. `npm install`
2. `npm run dev`
3. Base URL: http://localhost:5000

## Các vấn đề đang gặp
- Auth hoạt động nhưng refresh token chưa ổn.
- Middleware kiểm tra quyền chưa chắc chắn.
